CLASSES REFERENCE:
==================

STRUCTURE:
- blog-post (main article)
- post-header
- post-content
- post-section (main sections)
- post-intro (introduction section)
- post-navigation (TOC section)
- subsection (nested content)

TEXT ELEMENTS:
- post-title (H1 - only one)
- section-heading (H2)
- subsection-heading (H3)
- method-heading (H3/H4)
- lead-paragraph (intro text)

SPECIAL BOXES:
- quick-answer-box (summary/highlights)
- info-box info-box--learning (learning objectives)
- info-box info-box--features (feature lists)
- yellow-alert (warnings/alerts)
- quote-block (blockquotes)

LISTS:
- nomargin-list
- feature-list / feature-item
- stats-list
- step-list / step-item
- sub-step-list
- option-list
- toc__list

IMAGES:
- tutorial-figure
- tutorial-image
- tutorial-caption

TABLES:
- table-wrapper
- comparison-table
- table-caption
- table-header
- table-row-header
- table-cell

CODE:
- code-block
- code-block__header
- code-block__label
- code-block__copy-btn
- copy-btn__text
- code-block__content
- code-block__code

META:
- post-meta
- post-date
- post-reading-time

LINKS:
- External authority: target="_blank" rel="dofollow"
- External non-authority: target="_blank" rel="noopener"
- Internal: no target needed

IMAGES:
- First image: fetchpriority="high"
- Other images: loading="lazy"
- Always include: width, height, alt, title, style="aspect-ratio:W/H"
