How to Install Newsletter Plugin in WordPress

Installing the WordPress email newsletter plugin takes less than 2 minutes. Follow these steps:

Step 1: Access Plugin Menu

  1. Log into your WordPress admin dashboard
  2. Navigate to Plugins ‚Üí Add Plugin

Step 2: Search for Newsletter Plugin

  1. In the search box, type: "Newsletter awesome"
  2. Look for the plugin with 300,000+ active installations by Stefano Lissa

Step 3: Install & Activate

  1. Click "Install Now" button
  2. Wait for installation to complete (5-10 seconds)
  3. Click "Activate" button

Step 4: Initial Setup

After activation, you'll see a Newsletter menu item in your WordPress sidebar.

The plugin is now installed and ready to configure!

Creating Your First Subscription Form

Now let's add newsletter to WordPress by creating a subscription form.

Step 1: Access Form Settings

  1. Go to Newsletter ‚Üí Forms
  2. Click on Configure (‚öôÔ∏è Gear Icon) next to "Standard Form"

How to Embed Newsletter Form in WordPress

Setting Up Welcome Email

A welcome email is automatically sent when someone subscribes. You can customize it to match your brand.

Step 1: Access Welcome Email Settings

  1. Go to Newsletter ‚Üí Subscription
  2. Click on Welcome tab
  3. Open the Email tab
  4. Choose 1 of 3 options:
    • Default - uses the default email template. Yet you can still slightly Customize it if click on "Customize" below Welcome subject.
    • Composer - allows you to create a custom email. Prefered version.
    • Do not send - disables the welcome email

Step 2: Write Email Subject

  • Welcome to [Your Site Name]
  • Thanks for Subscribing, {name}

Step 3: Choose Email Editor

  • Default ‚Üí Customize ‚Üí Visual / Code Tab for HTML templates

How to Send Your First Newsletter

Now that your forms and emails are set up, let's send a newsletter! And there are actually two ways to do this:

Step 1: Create New Newsletter

  1. Go to Newsletter ‚Üí Newsletters
  2. Click "Add new" button

Step 2: Choose Template

You have two options:

  • Use a Template - Pre-designed layouts (10 of them)
  • Start from Blank - Empty canvas

Step 3: Write Your Newsletter

Subject Line:

Write a compelling subject (best practices):

  • Keep it under 50 characters
  • Use personalization: "Hi {name}, here's your update"
  • Create curiosity: "You won't believe what happened..."
  • Be clear: "Weekly Newsletter - January 2025"

Message Body:

  • Add header with your logo
  • Write your content in text blocks
  • Add images (optimized, under 200KB each)
  • Include call-to-action buttons
  • Add social media links in footer

Step 4: Click Next

Step 5: Select Recipients

Choose Your Lists:

  • Match all of them (sends to everyone)
  • Or select specific lists

Step 6: Schedule or Send

Option A: Send Now

  • Click "Send Now" button
  • Start real delivery? - Click "OK"

Option B: Schedule for Later

  • Click "Schedule"
  • Choose date and time
  • Click "Schedule" again and then confirm with "OK"

Step 7: Verify

  1. Check your own email inbox
  2. Verify the newsletter arrived
  3. Check formatting on desktop and mobile
  4. Click links to ensure they work

Congratulations! You've sent your first newsletter using the WordPress newsletter plugin.

Auto-Redirect After Subscribe

Automatically redirect users back to the main page or to a thank you page or special offer after they subscribe. Here is how to do it:

Step 1: Configure Redirect

  1. Go to Pages ‚Üí Newsletter ‚Üí Edit
  2. Click ‚úö Add icon to add new element to your page.
  3. Search for HTML element and click it.
  4. Copy and Paste this code inside HTML element:
    • Shortcode 
      <div class="thank-you-message">
  <h2>Thank you for subscribing!</h2>
  <p>Redirecting in <span id="timer">5</span> seconds...</p>
</div>

<script>
  // Only run redirect if we're on the confirmation page
  const urlParams = new URLSearchParams(window.location.search);
  const isConfirmed = urlParams.get('nm') === 'confirmed';

  if (isConfirmed) {
    let seconds = 5;
    const timer = document.getElementById('timer');
    const countdown = setInterval(() => {
      seconds--;
      timer.textContent = seconds;
      if (seconds <= 0) {
        clearInterval(countdown);
        location.href = '/';
      }
    }, 1000);
  } else {
    // Hide the thank you message if we're not on the confirmation page
    document.querySelector('.thank-you-message').style.display = 'none';
  }
</script>
  5. Click Save.

Step 2: Test

  1. Visit your subscription form.
  2. Subscribe with a test email.
  3. Verify redirect to home page works.

Troubleshooting Common Issues

Problem 1: Emails Not Sending

Solution:

  1. Check your WordPress email settings
  2. Install WP Mail SMTP plugin
  3. Configure SMTP (Gmail, SendGrid, etc.)
  4. Test email delivery

Problem 2: Form Not Appearing

Solution:

  1. Verify shortcode is correct: [newsletter_form]
  2. Check page isn't cached (clear cache)
  3. Ensure plugin is activated
  4. Try different page builder

Problem 3: Welcome Email Not Received

Solution:

  1. Check spam or junk folder
  2. Verify welcome email is enabled and saved
  3. Test with different email address
  4. Check email logs in Newsletter ‚Üí Help ‚Üí Logs

Problem 4: Slow Email Sending

Solution:

  1. Go to Newsletter ‚Üí Settings ‚Üí General Settings ‚Üí Sending
  2. Adjust emails per hour limit
  3. Consider upgrading hosting if needed
  4. Use external SMTP service for better delivery

Frequently Asked Questions

Is Newsletter plugin really free?
Yes, the Newsletter plugin is 100% free with unlimited subscribers and unlimited emails. There are premium addons available for advanced features, but the core plugin has everything most users need.
How many subscribers can I have?
Unlimited. Unlike MailChimp (500 free limit) or MailPoet (1,000 free limit), Newsletter plugin has no subscriber restrictions on the free version.
Can I use Newsletter plugin with Elementor?
Yes! You can add the newsletter form to any Elementor page using the Shortcode widget. Just paste [newsletter_form] into the widget.
Does it work with Gutenberg?
Absolutely. Use the Shortcode block in Gutenberg and paste [newsletter_form] to display the subscription form.
Is Newsletter plugin GDPR compliant?
Yes. The plugin includes privacy checkboxes, double opt-in confirmation, easy unsubscribe links, and data export/deletion features required for GDPR compliance.
Can I import subscribers from MailChimp?
Yes. Export your subscribers from MailChimp as a CSV file, then use Newsletter's import feature to upload them. Make sure to map the fields correctly (Name, Email).
Why are my emails going to spam?
WordPress's default email function can be unreliable. Install the WP Mail SMTP plugin and configure it with a proper SMTP service (Gmail, SendGrid, etc.) to improve deliverability.
Can I schedule newsletters?
Yes. When creating a newsletter, you can choose to send immediately or schedule it for a specific date and time.
How do I customize the form design?
You can customize the form using CSS. Go to Appearance ‚Üí Customize ‚Üí Additional CSS and add your custom styles for the form fields and buttons.
Can I send newsletters to specific groups?
Yes. Create multiple lists (Newsletter ‚Üí Lists) and assign subscribers to different lists. When sending a newsletter, you can choose which lists to send to.
Does Newsletter plugin slow down my website?
No. The plugin is lightweight and well-optimized. It only loads necessary resources on pages where the subscription form is displayed.
Can I track email opens and clicks?
Yes. The plugin includes analytics showing opens, clicks, bounces, and unsubscribes for each newsletter you send.
