How to Install Newsletter Plugin in WordPress

Installing the WordPress email newsletter plugin takes less than 2 minutes. Follow these steps:

Step 1: Access Plugin Menu

  1. Log into your WordPress admin dashboard
  2. Navigate to Plugins → Add New

Step 2: Search for Newsletter Plugin

  1. In the search box, type: "Newsletter by Stefano Lissa"
  2. Look for the plugin with 300,000+ active installations

Step 3: Install & Activate

  1. Click "Install Now" button
  2. Wait for installation to complete (5-10 seconds)
  3. Click "Activate" button

Step 4: Initial Setup

After activation, you'll see a Newsletter menu item in your WordPress sidebar.

The plugin is now installed and ready to configure!

Creating Your First Subscription Form

Now let's add newsletter to WordPress by creating a subscription form.

Step 1: Access Form Settings

  1. Go to Newsletter → Forms
  2. Click on ⚙️ Gear Icon next to "Standard Form"

Step 2: Configure Form Fields

Choose which fields to display:

  • ✅ Email (Required - cannot be disabled)
  • ✅ First Name (Recommended)
  • ☐ Last Name (Optional)
  • ☐ Gender (Optional)
  • ☐ Privacy Checkbox (Required for GDPR)
**Step 3: Customize Labels** Edit the text that appears in your form: - Button text: "Subscribe" (or "Join Our Newsletter") - Placeholder text: "Enter your email" - Success message: "Thank you for subscribing!" **Step 4: Copy Shortcode** At the top of the settings page, you'll see: ``` [newsletter_form] ``` Copy this shortcode - you'll need it in the next step. **Step 5: Save Settings** Click **"Save"** at the bottom of the page.
